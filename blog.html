Question-1: What are the different ways to select an element in the DOM?

1. document.getElementByTagName()
2. document.getElementById()
3.document.getElementByClassName()
4. document.querySelector()
5. document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes HTML, innerText gets visible text, and textContent gets all text without HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and propagates up to its parent elements in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement() to create, appendChild() or append() to add, and removeChild() or remove() to remove elements in JavaScript.
